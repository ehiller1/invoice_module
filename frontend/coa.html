Import Chart of Accounts (Spreadsheet)

Upload an Excel (.xlsx, .xls) or CSV file with accounts and/or funds. See templates below for column requirements.

Show Accounts spreadsheet template

CSV/Excel format (one row per account):

Account Number,Account Name,Account Type,Fund,Is Active
1000,Cash - Checking,ASSET,GEN,true
2010,Accounts Payable,LIABILITY,GEN,true
5010,Clergy Salary,EXPENSE,GEN,true
7100,Utilities,EXPENSE,GEN,true
8310,World Service Fund,EXPENSE,GEN,true

Columns: account_number (required), account_name (required), account_type (Asset/Liability/Equity/Revenue/Expense), fund (optional), is_active (true/false, default true)

Show Funds spreadsheet template

CSV/Excel format (one row per fund):

Fund ID,Fund Name,Category,Restriction Class
GEN,General Operating,GENERAL_OPERATING,WITHOUT_RESTRICTION
BLD,Building Fund,CAPITAL_CAMPAIGN,WITH_RESTRICTION_PURPOSE
END,Endowment,ENDOWMENT,WITH_RESTRICTION_PERMANENT

Columns: fund_id (required), fund_name (required), category (optional), restriction_class (optional)
